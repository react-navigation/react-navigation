# Example for React Navigation

If you want to run the example from the repo,

- Clone the repository and run `pnpm install` in the project root
- Run `pnpm example start` to start the packager
- Build and install the development build with `pnpm example android` or `pnpm example ios`
- Follow the instructions in the terminal to open the app

You can also run the currently published app on Web: <https://react-navigation-example.netlify.app>
